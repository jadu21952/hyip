WealthX Demo Simulation Website

This ZIP contains a static HTML/CSS/JS demo UI only.
It does not include real deposits, real withdrawals, payment gateway, wallet, bank, crypto collection, or backend.
Use it only as a front-end simulation preview.

Files:
- index.html
- style.css
- script.js

How to run:
1. Extract ZIP
2. Open index.html in browser
3. Or upload all files to GitHub and deploy with Cloudflare Pages
